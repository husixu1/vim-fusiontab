# coc-exposejump

Expose some internal states of coc.nvim.

## Install

`:CocInstall coc-exposejump`

## Keymaps

`nmap <silent> <C-l> <Plug>(coc-coc-exposejump-keymap)`

## Lists

`:CocList demo_list`
